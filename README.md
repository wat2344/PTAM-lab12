# PTAM-lab12

Данная лабораторная работа посвящена изучению специализированного текстового редактора Vim.

## Tutorial

```bash
vimtutor ru   # запуск туториала на русском языке
```

В процессе выполнения встроенного учебника были освоены следующие команды и приёмы работы в Vim:

### Навигация

- h j k l — перемещение курсора влево, вниз, вверх, вправо.
- w — переход к началу следующего слова.
- b — переход к началу предыдущего слова.
- e — переход к концу слова.
- 0 (ноль) — переход в начало строки.
- $ — переход в конец строки.
- gg — переход в начало файла.
- G — переход в конец файла.
- Ctrl+f / Ctrl+b — прокрутка страницы вперёд/назад.

### Редактирование текста

- x — удаление символа под курсором.
- i — переход в режим вставки перед курсором.
- a — переход в режим вставки после курсора.
- o — вставка новой строки снизу и переход в режим вставки.
- dw — удаление слова от курсора до его конца.
- de — удаление от курсора до конца слова (включая последний символ).
- d$ — удаление от курсора до конца строки.
- dd — удаление целой строки.
- u — отмена последнего действия (undo).
- Ctrl+r — повтор отменённого действия (redo).
- p — вставка ранее удалённого или скопированного текста после курсора.
- yy — копирование (yank) строки.
- yw — копирование слова.
- r — замена одного символа без выхода из нормального режима.
- ce — изменение до конца слова (удаление и переход в режим вставки).
- . — повтор последнего изменения.

### Поиск и замена

- /pattern — поиск вперёд по тексту.
- ?pattern — поиск назад.
- n / N — повтор поиска в том же / противоположном направлении.
- :%s/old/new/g — замена всех вхождений old на new во всём файле.

### Работа с файлами и окнами

- :w — сохранение файла.
- :q — выход.
- :wq — сохранение и выход.
- :q! — выход без сохранения.
- :split / :vsplit — разделение окна горизонтально / вертикально.
- Ctrl+w + стрелки — переключение между окнами.

### Настройки

- :set number — включение нумерации строк.
- :set hlsearch — подсветка результатов поиска.

### Сохранение туториала

